# npm

Descarga las dependencias del proyecto
```bash
-npm install
```

Monta en modo desarollo mi app para probarla
```bash
-npm run dev 
```

Crea la aplicacion para meterlo en un servidor de verdad
```bash
-npm run build 
```

Para instalar una dependendia en particular
```bash
npm instal tone
```
